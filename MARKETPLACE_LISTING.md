
# Marketplace Listing – ReleaseScribe (Changelog Generator)

**Elevator pitch**  
Generate **CHANGELOG.md**, bump **SemVer**, and create **GitHub Releases** from **Conventional Commits**—automatically.

**What it does**
- Detects next version (major|minor|patch)
- Generates `CHANGELOG.md` (Keep‑a‑Changelog style)
- Creates GitHub Release with human‑readable notes

**Standards**
Conventional Commits, SemVer, Keep a Changelog.

**Permissions**
`contents: write` (to update changelog and create releases).
